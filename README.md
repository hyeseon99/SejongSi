# SejongSi 프로젝트

### 🌟 프로젝트 소개
- **프로젝트 명:** 세종엔에서 만나는 AI와 빅데이터를 활용한 개인 맞춤형 병원·음식점 추천 시스템
- **개발 목표:** 
  - 세종시 공모전에 제안된 데이터를 기반으로 사용자 맞춤형 추천 시스템 개발
  - 공공 데이터 활용 가능성을 증대
- **주요 역할:**
  - 연령대별 맞춤형 음식점 추천 시스템 개발
  - **Streamlit**을 활용한 데이터 시각화 및 웹페이지 구현
- **성과:** 프로젝트 완수 및 공공 데이터를 활용한 창의적 접근으로 **세종시 공모전 참가**
